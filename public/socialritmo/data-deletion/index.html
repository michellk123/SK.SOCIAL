SocialRitmo — Data Deletion

Last updated: 4 June 2026

How to delete your Instagram data from SocialRitmo

SocialRitmo (a product of S.K Social) only stores Instagram data if a social media manager has connected an Instagram account to display its recent posts inside their planning grid. You can have that data deleted at any time using any of the methods below.

Option 1 — Remove SocialRitmo in Instagram (automatic)

  1. Open Instagram → Settings → Apps and Websites (or Website Permissions).
  2. Find SocialRitmo and choose Remove.

Instagram notifies SocialRitmo automatically. We then immediately delete the stored access token and Instagram identifiers for that connection. No further action is needed.

Option 2 — Disconnect inside SocialRitmo

If you manage the account inside SocialRitmo, open the client’s content screen and tap Disconnect. This deletes the stored token and Instagram identifiers for that connection right away.

Option 3 — Email request

Email saraik.social@gmail.com from the address associated with your account, or include the connected Instagram username, and ask us to delete the connection’s data. We will process the request and confirm within 30 days.

What gets deleted

For the affected connection we delete:

  • the encrypted Instagram access token,
  • the stored Instagram user ID and connection timestamps.

SocialRitmo does not store copies of the Instagram account’s posts — post thumbnails are displayed live from Instagram and are not retained after deletion.

Confirmation

When Instagram triggers an automatic deletion (Option 1), SocialRitmo returns a confirmation code and this status page. If you reached this page with a ?id= value in the address bar, that is your confirmation code — keep it for your records. To verify the status of a request, contact saraik.social@gmail.com with that code.
